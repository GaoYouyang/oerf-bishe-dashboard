唯一主线同精度降成本

C · Physics-corrected operator warm start

输入多视角 BOS 观测与几何，学习生成三维初值 x0；随后使用与经典基线完全相同的物理迭代器精修到同一误差阈值。

主指标
部署主表报终点等价、线性 A/AT 或非线性 forward/JVP/VJP、wall time 与失败尾部；oracle time-to-target 单列。
强基线
zero、反投影 ATy、PCGLS/CGLS、简单插值或阻尼，以及 DeepONet/FNO warm start。
首个候选
Adjoint-Residual FNO：线性阶段复用一次 ATy；非线性阶段使用参考态 JTr，避免特征本身先消耗大量物理调用。
保留为审计不是独立选题

B → C 的数值卫生门

先确认 forward、adjoint、单位、mask 与 curved/straight ray 层级，否则 warm start 的“加速”可能只是错误算子上的假改善。

必须通过
forward replay、伴随一致性、单位表、输入输出 shape 与成本计数。
失败处理
审计失败就修接口，不把模型结果写进论文主结论。
压力测试不单独立题

A → C 的少视角外推门

相机数、视角布局或噪声变化时，检查 warm start 是否仍能减少成本；不能只报告训练分布内的平均提升。

最小测试
留出功率、火池尺寸或时间轨迹，并改变视角数与噪声水平。
停止门
若尾部失败率上升或最终精度受损，回退到反投影初值。
后续升级当前冻结

D · 单帧成立后再进入 4D

只有 C 在独立三维轨迹上通过同精度成本门，才考虑用前一帧、低秩时间基或演化算子继续降低连续帧成本。

启动条件
单帧 C 有真实可重复优势，且时序数据合同、同步与几何保持方式已明确。
当前动作
不训练 4D 网络，不提前增加模型复杂度。
数据不是一次性索取

按四级台阶接触组内材料

每一级都有明确用途。上一层没有跑通时，不扩大权限或复制更多数据。
