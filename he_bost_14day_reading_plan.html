两周后的合格状态

你应该能用 5 分钟解释：BOS/BOST 观测链条是什么，为什么 NeRIF 用神经隐式折射率场替代体素，传统 baseline 应该怎么设，PIV-BOST 的补偿可能发生在哪一层，4D BOST 为什么要低秩/张量分解，以及你第一月能交付什么。能讲到这个程度，再去找师兄就不是“求方向”，而是在给师兄一个可裁剪的项目方案。

固定日输出

1 个公式

写出变量含义和近似条件，例如 rho -> n -> deflection -> displacement 或重投影误差。

1 张图/表

可以是论文图复刻、变量链图、baseline 对比表、数据字段表或失败样本图。

1 个动作

小 demo、伪代码、参数矩阵、下载/入库、或一段 notebook，不要求每天都很大。

1 个问题

问题要能让师兄作判断，例如 baseline、数据字段、补偿层级、公开边界或第一月交付。

PIV/4D/会前包

5 UBOST: Fast and robust volumetric refractive index measurement by unified BOST。
10.1007/s00348-020-2912-1		 它作为强传统 baseline 的 forward model、robustness language、实验评价方式。 写一页 UBOST 精读卡：它强在哪里，NeRIF 不能只赢一个弱 toy。 组内更认可 UBOST、SIRT/Landweber，还是已有代码作为 baseline？
6 Direct BOST using radial basis functions。
10.1364/OE.459872		 连续表示前史：RBF 参数化、直接重构、连续场和平滑先验。 做一张 voxel / RBF / MLP neural field 对照表。 如果不能复现 UBOST，RBF/direct BOST 是否能作为论文中的方法邻居而非实验 baseline？

PoF DOI / arXiv		 坐标输入、network 输出、折射率场、loss、projection/displacement consistency。 写 NeRIF-style 最小伪代码，不求完整复现，只要变量流正确。 NeRIF 里最应该本科复现的是 loss、network、少视角实验，还是真实数据接口？
8 NeRIF experiments: numerical validation、real flame、noise/view/mask 相关结果。 评价指标：L2/SSIM/CC、重投影误差、切片图、视角数、噪声水平、模型容量。 列出 M0/M1/M2 实验矩阵：view count、noise、encoding、hidden width、mask。 真实组内结果更看重重构场误差、重投影误差、火焰物理解释，还是速度补偿效果？
9 NeRIF reproduction plan + neural fields basics: Fourier features / SIREN / NeRF / Instant-NGP。 encoding 为什么重要；高频场、平滑性、过拟合、regularization。 做一个 2D/3D coordinate MLP 拟合 phantom 的最小 demo 计划。 师兄是否允许用 simplified NeRIF-style demo，而不复刻论文全部工程细节？
10 NeDF / neural deflection fields。
PoF DOI / arXiv		 NeDF 和 NeRIF 表示对象不同：deflection/gradient 相关场 vs refractive-index field。 写一张 NeRIF vs NeDF 表：primitive、loss、输入、输出、适合数据和局限。 师兄希望我把 NeDF 写成对照竞品，还是只放 related work？
11 NRIP: Neural Refractive Index Primitives。
DOI / arXiv		 hash-like encoding、3D mask、automatic/discrete gradient loss、真实 flame BOS 评价。 把 NeRIF 的可做消融扩展到 encoding / mask / gradient loss 三类。 NeRIF 方向是否需要引入 NRIP 的 mask/gradient-loss 思路做本科创新点？
12 Simultaneous PIV-BOST 2025。
10.1007/s00348-025-04093-y		 折射率场如何服务速度测量补偿；摘要/图/方法里看补偿位置和评价指标。 准备一个 synthetic particle image distortion toy 的任务书，作为无真实数据时的 B 路线降级。 能否给同步 PIV-BOST 的最小样例？如果不能，图像层 synthetic toy 是否对组里有用？

ACM 正式全文 · 结构化精读		 4D 的核心不是“多跑几帧”，而是 MM plane-pairs、低秩时序共享、DC/forward correction、频谱保真和混合精度共同作用。 读 Figure/Table/Appendix 后做 M3B 计划：rank-noise-view-frame-bias scan；单独抄出正文的 L2 定义、100G0 DC 消融和 13.08 GB 显存表。 4D BOST 拆给本科生时，最有价值的是 rank scan、forward-model bias、数据健康报告还是 temporal/frequency metric？正文与公开 config 的版本差异应如何解释？

三条路线的 14 天后决策

A: NeRIF/BOST 鲁棒性

默认锁定

如果 14 天后你能跑通 phantom + baseline + simplified coordinate MLP，就把毕设题目写成“少视角 BOST 的神经隐式折射率场重构与鲁棒性分析”。

B: PIV-BOST 补偿接口

有数据再升

如果师兄给同步数据或明确需要图像层/速度层工具，就把 B 作为第二阶段；否则只保留 synthetic particle-image toy，不承诺完整 PIV-BOST。

C: 4D BOST 子模块

挑战但要收缩

如果师兄明确希望你接 4D，就只做 low-rank / temporal metric / report 子问题，不说“完整复现 TOG”。

不要做的事

  • 不要把两周计划写成“读完所有论文”。真正要交付的是公式、图、demo 和可问的问题。
  • 不要等真实数据来了再开始。合成数据闭环是本科毕设的保险。
  • 不要把 WebVPN 下载的出版社 PDF 放进公开 GitHub Pages 或公开跳转页；本机 `private_library/` 只服务个人学习。
  • 不要一开始承诺完整 PIV-BOST、完整 stereo-PIV 或完整 4D BOST。先做可裁剪的子模块。
