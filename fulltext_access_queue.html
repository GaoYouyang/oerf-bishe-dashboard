如何使用

P0 用来支撑你的毕业设计主线判断：何远哲 PIV-BOST / 4D BOST、蔡组 direct BOST、Grauer UBOST、BOS 灵敏度和 PIV 折射率误差。4D BOST 已通过 ACM HTML/eReader 解决正式全文阅读，不再需要 WebVPN；它保留在原始队列里用于记录状态。P1 用来补传统基线与误差评估，P2/P3 用于旁支和全景背景。

下载后的 PDF 只放本机 private_library/inbox/ 或 private_library/pdfs/，运行本地同步脚本后进入私有库；公开 GitHub Pages 不上传这些 PDF。

如果浏览器把文件直接保存到系统下载目录，可以用本地同步脚本扫描最近下载的候选文件：python3 private_library/tools/sync_private_library.py --scan-downloads --max-scan-files 20。匹配成功的 PDF 会复制到本机私有库，公开网页只保留状态和 DOI。
