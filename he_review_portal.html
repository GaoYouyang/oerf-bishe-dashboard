总判断

当前建议工作题：固定相机预算下稀疏视角 BOST 的强数值-残差逆算子初始化与 NeRIF 物理细化。

已有证据：固定 168-sample dataset、三个优化种子中，43,363 参数 Residual FNO 在五域 field/held-out 的 30/30 个 seed-domain 单元优于 49,111 参数 U-Net；IID field mean 为 0.2187 vs 0.3000。

关键反例：Residual FNO 在 IID/noise OOD 三种子全胜，Absolute FNO 在 3-view/joint OOD 三种子全胜；family OOD 仍约 0.67。固定 residual skip 不能当普适答案。

最新负对照：固定 view gate 明显变差；metadata gate 只把五域 field oracle regret 从 3.10% 降到 3.08%，held-out regret 从 1.65% 升到 2.00%。因此不再主张“一个标量门控”足够。

双分支与零空间证据：独立专家的 support-fit 闭式融合已到达 endpoint-oracle-or-better 的 79.17%；事后精确 null oracle 显示 support-fit 之上仍有 38.63% 平均 field 改善空间。但直接学满幅 null corrector 的总体改善为 -0.126%，这个负结果保留作为下一步设计约束。

被淘汰的 v2d 路径：固定 Q_fit=80°、Q_audit=60°；88 个独立体场先汇总 3 个 model seeds，再做 20,000 次五来源域等权 bootstrap。fixed learned correction 相对同重建预算 direct 在 K=4/6/8 分别为 -20.17% / -13.07% / -15.11%，当前路径 0/3 通过。

最新 v3a 候选：每个 K 独立训练并匹配相机 mask 后，FBP-lift U-Net/FNO 仍全部失败；ridge-initialized residual FNO 相对 validation-locked ridge 在 K=4/6/8 改善 +21.54% / +19.68% / +16.91%，p10 均为正，伤害率为 0% / 0% / 4.17%，Q_audit 残差同步改善。

自有算法 v3b：U-Net/FNO/DeepONet/自有模型获得相同 42-channel ray/mask/angle/ridge 输入。自有 45,973 参数 ray-set 增强候选相对 validation-locked FNO 在 K=4/6/8 平均 +1.77% / +5.36% / +1.09%，只有 K=6 通过 development gate；K=4/K=8 各有一个种子转负，thin-front OOD 尚未胜出。

v3c 协议闸门：zero-init adapter 包装冻结 44,203 参数 FNO，新增 4,988 个可训练参数。36 个分层样本的初始输出差为 0.0，三次 optimizer steps 后底座漂移为 0.0；328 个 dev2 fields 与 v3b 的 seed/field/observation hashes 重叠均为 0。这不是性能结果。

v3c K=6 dev2 性能判决：三种子同 checkpoint 分叉后，adapter 相对 continued FNO 的 field superiority 为 -4.965%，95% CI [-5.569%, -4.397%]，74.219% 体场退化超过 1%；adapter 相对 base FNO 有 +1.425%，但 continued FNO 相对 base 为 +5.846%。adapter 训练/推理约慢 7.18×/6.46×。

v3d FNO plateau 审计：红队复核后 validation 改为每个三维场等权。K=6 FNO 串接到 96 epochs 为 0.16646→0.11746，末 block 仍改善 2.81% / 2.96%，高于 plateau 阈值；复用 dev2 field 相对 epoch 24 为 +16.53%，但不参与本轮停止点，也不是新的全项目盲集。

v3d 240-epoch optimizer protocol 审计：batch-order contract 绑定实际 train indices；continuation optimizer 在 base 后新建。`carry continuation-Adam + restart cosine` 仍是 validation 冠军（0.094139）但未 plateau；long-cosine 从 epoch 192 plateau，最终弱 1.66%。复用 dev2 上冠军相对 base field 改善 +22.09%，但 family OOD 轻微反向。

v3e-v3g 跨架构与容量账本：fresh-worker 成本为 FNO 14.72M FLOPs-v1、1.49/8.54 ms，DeepONet 13.66M、0.77/3.01 ms；adapter 4,988 可训练参数但训练步约为 FNO 的 11.0×。v3f 中 FNO/rank-48 DeepONet epoch-240 validation 为 0.09414/0.17573。v3g 再做 8 架构 × 3 学习率 × 3 种子的 72-run screen；短程 rank-64 冠军延长后为 0.17609，复用 dev2 相对 reference -0.16%、CI 跨 0，因此不替换 rank 48。下一版建议 FNO 做空间主干、set branch 只做 acquisition conditioning。

v3h-v3i 几何可辨识性与数据合同：旧 K=6 数据的 328 样本共用一布局，correct/shuffled geometry 无法被识别；28 布局在同 40 个 validation 场上使 ridge 平均场误差跨度达 64.95%。v3i 已将 328 个唯一场均衡分配到 28 布局，训练 16 布局各 10 场，validation 使用 4 个未见布局；Q_audit 零通道、共享噪声与私有 NPZ 哈希均通过独立 validator。只授权五组三种子功能 pilot。

v3j GC-SRO 机制负结果：12 次等参数训练后，static adapter 相对 locked FNO validation 改善 +3.6336%，但 correct geometry 相对 static/K-cardinality/shuffled 仅 -0.00019%/-0.00008%/-0.01628%，三组都只有 1/3 种子正向且 CI 跨 0。同模型 descriptor swap 显示 embedding 能区分布局，但 correction 仅变 0.1785%，最终场收益 -0.00015%。因此停止当前全局几何调制的扩参。

v3k-A 等曝光反事实判决：M1 固定一个布局重复 4 次，M4 同一场使用 4 个不同布局；两臂均为 160 个独立场、640 行、相同优化步与 1,023 个可训练参数。M4 correct 相对 shuffled validation 为 +0.0675% [95% field-cluster CI +0.0410%, +0.0952%]、3/3 seeds 正向；M4-M1 interaction 为 +0.0349% [+0.0126%, +0.0580%]。信号虽恢复，但 CI 上界远低于预注册 +0.25%。同模型 correction swap 已升至 8.1399%，最终场收益仅 +0.0130% [-0.0138%, +0.0391%]，因此停止全局 geometry-to-channel modulation 的容量搜索。

v3k-B 体素射线集机制判决：12 次等参数训练中，correct ray-set 相对 geometry-only 改善 +1.4445% [95% field-cluster CI +0.5149%, +2.3801%]、3/3 seeds 正向，说明局部 ray values 有用；但相对同 mask、同 ray values、同 angle multiset 的 shuffled-pairing 为 -0.1967% [-0.4023%, -0.0075%]。同模型 pairing swap 使 correction 变化 19.0683%，最终场收益却为 -0.3586% [-0.6490%, -0.1108%]、0/3 seeds 正向。因此停止 attention 扩参，下一轮只检验 forward/adjoint 定义的局部数据一致性残差。

v3k-C 伴随方向数值闸门：28 个 mask 的 adjoint identity 最大误差 1.94e-15，finite-difference gradient 最大误差 2.44e-8。修复非幂等 soft-support 混杂并从 validation selection artifact 移除 Q_audit 后，只用 40 个 validation fields 选中 geometry-normalized standard Landweber β=1.9,T=64；相对 feasible FNO 的场级收益为 +44.6024% [95% CI +40.1023%, +49.1099%]。四个 development test domains 的最小平均收益为 +22.4001%，但最差 noise-OOD 布局仍有 9.375% 场伤害超过 1%。结论是 Landweber 成为必须击败的强基线，不是项目创新。

v3k-D 强数值控制判决：用 validation 分别选择 raw/feasible ridge、global-step、geometry-step、spectral lookup 和逐场二次最速下降，并在读取 development test 前写出 selection commit。FNO-start quadratic 的 validation relative L2 为 0.143855，相对 feasible FNO 改善 +45.5258% [95% CI +41.3505%, +49.6840%]；但相对 128-call geometry Landweber 仅 +1.2443% [95% CI +0.5282%, +1.9583%]，需把总算子调用从 128 提到 193。该小增益在 noise-OOD 反向 -0.9128%，53.125% 体场退化超过 1%；逐步目标函数上升率为 0。learned scalar 因 projected BB 和 fresh case-level lock 尚未完成而保持关闭。

几何外推边界：M3B 已完成 1,728 个观测单元的 nested leave-one-geometry-out。固定 rank 3 的 mean regret 为 1.918%，无额外 held-out 的 LOGO selector 为 0.273%；当前 combined UQ 与 full-rank fallback 都不足以支持系统拒答。
