Neural Field / Encoding 到 NeRIF 的桥梁

这页专门解释 Fourier Features、SIREN、NeRF、Instant-NGP、DeepSDF、Occupancy Networks 和 Neural Fields survey 为什么值得读，以及它们如何服务何远哲方向的 NeRIF / NRIP / NIRT / 4D BOST。核心结论很简单：它们是表示法和编码基础，不是 BOST baseline。

总判断

给师兄看的说法：我补这些文献不是想把题目改成 NeRF，而是为了把 NeRIF 里的 coordinate neural field、positional/Fourier encoding、hash encoding、连续场和导数监督讲清楚。真正的观测模型、物理量和评价仍然围绕 BOST 的折射率场、位移场和重投影残差。

NeRIF 的困难不只是“用网络拟合三维函数”，而是用很少视角、带噪声的 BOS 位移观测去恢复一个有强梯度、局部高频、物理约束和投影一致性的折射率场。这里的 neural field 文献解决的是表示能力和优化偏置问题。

七篇怎么读

论文 该抽什么 接到 NeRIF / NRIP 的位置 本地入口
Fourier Features
NeurIPS 2020 / arXiv 2006.10739		 普通 MLP 的 spectral bias、Fourier feature mapping 如何让低维坐标学高频函数。 解释为什么折射率场的局部强梯度、火焰边界和小尺度结构不能只靠 plain MLP。 本地 PDF / 官方页
SIREN
NeurIPS 2020 / arXiv 2006.09661		 周期激活、连续信号、导数保真、初始化。 帮助理解从场值到梯度、偏折和 projection loss 的可微监督；适合对照 NeDF/NRIP 的梯度和偏折场表示。 本地 PDF / 官方页
NeRF
ECCV 2020, DOI 10.1007/978-3-030-58452-8_24		 coordinate MLP、positional encoding、ray integration、多视角监督。 用于区分 NeRF-style radiance rendering 与 BOST refractive-index tomography：一个是颜色/密度渲染，一个是折射率/偏折/位移一致性。 本地 PDF / DOI
Instant-NGP
ACM TOG 2022, DOI 10.1145/3528223.3530127		 multiresolution hash grid、速度、显存、高频细节。 NRIP 已明确用 hash encoding；这篇是解释 hash encoding 为什么能加速和增强高频表达的根文献。 本地 PDF / ACM
DeepSDF
CVPR 2019, DOI 10.1109/CVPR.2019.00025		 连续标量场、坐标到值的隐式函数、离散体素外的表示方式。 只借“连续标量场”的思想，帮助把 n(x,y,z) 当作连续函数；不要借 shape reconstruction 结论。 本地 PDF / CVF
Occupancy Networks
CVPR 2019 / arXiv 1812.03828		 function-space 3D reconstruction、连续坐标采样、体素/网格替代。 作为 optional 背景，说明 neural field 是一类函数空间表示；不需要精读所有 shape 实验。 本地 PDF / CVF
Neural Fields Survey
Computer Graphics Forum 2022, DOI 10.1111/cgf.14505		 术语总览：coordinate networks、encodings、signals、rendering、physics extensions。 用于写 related work 的总框架：NeRIF / NeDF / NRIP / NIRT 都可放在 neural-field inverse problem 下面，但物理观测不同。 本地 PDF / DOI

概念映射

未知量

NeRF 常把 scene 表示成 radiance/density；DeepSDF 表示 signed distance；NeRIF/NRIP 要表示的是折射率场或折射率相关 primitive。写论文时要把 unknown variable 写清楚。

观测量

NeRF 用图像颜色监督；BOST 用背景图案位移、偏折或重投影残差监督。两者都沿光线积分，但 measurement equation 完全不同。

编码

Fourier/SIREN/hash grid 解决的是坐标网络表达高频和训练速度的问题。它们可以进入 ablation，但不能替代 BOS forward model。

可以变成的本科实验

实验 最小做法 能回答的问题 风险边界
Encoding ablation 在现有 M1/M2 synthetic BOST toy 上比较 plain MLP、Fourier features、SIREN-style activation、hash/grid-like features。 少视角、噪声和高梯度 phantom 下，哪种表示更稳，是否减少 oversmoothing。 hash/grid-like toy 可以轻量模拟，不必复现完整 Instant-NGP CUDA 实现。
Capacity scan 固定 forward model，扫描层数、宽度、频率带宽、正则化权重。 NeRIF 的性能瓶颈是网络容量、视角数、噪声，还是 forward-model mismatch。 synthetic 结论不能直接当真实火焰结论，需要真实数据或师兄认可。
Derivative loss 对照 比较只拟合场值、拟合梯度、拟合投影/偏折残差，观察边界 artifact。 NeRIF/NeDF/NRIP 到底是在恢复 n、grad n，还是只保证 image displacement 一致。 需要把物理单位和离散差分写清楚，否则容易变成纯图像损失。
Runtime / memory report 对同一个 phantom 记录训练时间、显存、重投影误差、PSNR/SSIM 或 L2。 如果师兄关心 4D 或高速重构，encoding 的速度收益是否值得。 本科阶段先报本机/单 GPU 情况，不夸大到实验室高速系统。

阅读顺序

  1. 先读 NeRIF 正文和 appendix，圈出它的输入、输出、loss、编码和评价指标。
  2. 读 Fourier Features，只抽 spectral bias 和 positional encoding 的核心图。
  3. 读 SIREN，只抽导数监督和连续信号表示。
  4. 读 Instant-NGP，只抽 hash encoding 的速度/显存/高频细节权衡。
  5. 回到 NRIP，看它为什么选 hash encoding、mask 和 primitive 表示。
  6. 最后用 Neural Fields survey 统一术语，DeepSDF/Occupancy Networks 只作 optional 背景。

下次问何远哲

  • NeRIF 组内 baseline 是否允许我做 encoding ablation，还是应先完整复现论文设置？
  • 真实数据里最痛的是少视角、噪声、高梯度 oversmoothing、训练慢，还是几何/标定误差？
  • 如果只做本科题，师兄更希望看到 plain MLP vs Fourier/hash 的小实验，还是先要 data loader 和重投影验证？
  • NRIP/NeDF 是否需要正式写进 related work，还是只作为会前讨论的外部竞品？
  • 4D BOST 如果关注速度，是否允许把 hash/grid encoding 作为未来加速方向写在展望里？

写作边界

不要写成：NeRF 可以直接解决 BOST。

更准确的写法是：NeRF、SIREN、Fourier Features 和 Instant-NGP 共同说明了 coordinate neural fields 的表达和编码机制；NeRIF 的贡献在于把这种表示放入 BOST 折射率场的物理 forward model、稀疏视角约束和重投影验证中。
