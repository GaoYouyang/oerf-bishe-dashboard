脑中地图

读任何一行时，都先问它在哪一层

数据层

像素坐标、相机位姿、时间和真实 u/v 位移被整理为一批光线。

  • rays [B,6]
  • displacement [B,2]
  • time [B,1]

场与物理层

MMmodel 查询 n(x,y,z,t)，TD_Base 在六个邻域点做有限差分，再沿光线积分。

  • 低维平面压缩四维场
  • 中心差分得到三维梯度
  • W2C 投影为图像位移

训练层

预测位移与真实位移比较，误差通过自动求导回到张量平面和 MLP 参数。

  • MSE 数据误差
  • TV / 边界 / 路径约束
  • backward + Adam 更新
