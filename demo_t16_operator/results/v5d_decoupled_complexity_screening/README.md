# v5d decoupled-complexity screening

This package is post-open development on v5c, not a new lock. Truth labels are used to rank variants. The complete radius/fold/kappa surface is saved so later analyses do not silently rebuild or retune it.

Top development variant: `morozov_0p50` with 5/6 nearest-bank matches. That number is not confirmatory and cannot authorize a paper claim.
