# PSU-C1 scale-separated figure

This is a derived visualization of the sealed, independently validated PSU-C1
summary. It does not alter any metric, gate, method, split, or decision.

The top row zooms the bounded first-direction controls so that their roughly
one-percent signal can be compared with the preregistered five-percent oracle
gate. The lower row separates active-view residual, held-out-view residual, and
the strong inverse-Sobolev comparator. The last panel makes the key tension
visible: inverse-Sobolev substantially improves analytic field error while
often increasing the active measured residual.

The figure is post-open descriptive evidence on public PSU ray geometry with
analytic fields and synthetic noise. It is not real experimental truth, unseen
rig generalization, algorithm superiority, novelty, or a breakthrough.
