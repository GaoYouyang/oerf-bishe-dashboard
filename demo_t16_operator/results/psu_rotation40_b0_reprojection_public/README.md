# PSU rotation-40 frozen-B0 development baseline

This public folder contains only an aggregate diagnostic rendered from
`docs/psu_rotation40_b0_reprojection_public_summary.json`. It contains no raw
measurements, masks, rays, geometry arrays, predictions, or reconstructed
voxels.

The figure is a development-baseline result, not a three-dimensional accuracy
result and not an algorithm-superiority result. The final-audit rotations remain
sealed.
