# 毕设选题决策矩阵

用途：和何远哲沟通前，把备选题压缩成 2-3 个，而不是“我什么都想做”。

## 评分规则

每项 1-5 分，总分不等于必须选择。主线贴合和可毕业性权重最高。

- 主线贴合：是否贴 BOST / NeRIF / PIV-BOST / 4D BOST。
- 可毕业性：三个月内是否能做出最小 demo，一年内是否能写成论文。
- 组内价值：是否能帮课题组处理数据、复现实验、出图或做工具。
- 数据可启动性：没有真实数据时能不能靠合成/开源数据先跑。
- 新意：是否有鲁棒性、误差传播、时序先验或真实数据迁移。
- 风险：分数越高代表风险越低。

## 推荐组合

最稳组合：

- 主标题：少视角 BOST 的神经隐式折射率场重构与鲁棒性分析。
- 第一阶段：合成数据 + 传统 baseline + 简化 NeRIF。
- 第二阶段：接真实 BOST 样例或开源 BOS/TBOS 数据。
- 加分副线：E 的时间同步/缺失视角误差图谱 + F 的位移/光流质量控制工具。

最贴师兄组合：

- 主标题：面向 BOST/NeRIF 的可复现重构、误差分析与真实数据迁移。
- 第一阶段：照 NeRIF 做最小闭环。
- 第二阶段：根据师兄需求选择 PIV-BOST image-pair 补偿、PIV velocity-field error propagation、BOS displacement quality report 或 4D BOST 子问题。

最有论文感组合：

- 主标题：少视角 BOST 神经隐式重构中的视角、噪声与坐标编码鲁棒性研究。
- 创新点：不是提出大模型，而是系统量化什么时候 NeRIF 比 voxel baseline 更稳。
- 图表：视角数曲线、噪声曲线、坐标编码消融、NRIP-style hash-like encoding 对照、重投影一致性、时间/显存曲线。

最贴真实实验误差组合：

- 主标题：面向火焰 BOST 三维重构的时间同步误差、缺失视角与投影补全鲁棒性研究。
- 创新点：不只比较算法精度，而是回答真实实验里“这组多视角数据还能不能可信重构”。
- 图表：time offset / missing view / noise 三维热图、投影补全对比、同步误差容忍范围、自动质量报告。

最稳工具交付组合：

- 主标题：面向 PIV-BOST/BOST 的图像位移估计、光流 benchmark 与质量控制工具。
- 创新点：不声称替代 NeRIF，而是给真实 BOST/PIV-BOST 数据进入重构前增加 input health check。
- 图表：Farneback/DIS/OpenPIV/RAFT-lite 位移误差表、bad-region mask、wavefront/atmospheric-turbulence stress test、实时/离线处理耗时对比。

## 放弃规则

- 如果题目必须完整搭实验台才能开始，暂时放弃。
- 如果没有真实数据就完全不能推进，暂时降级。
- 如果 PIV-BOST 只能拿到最终速度场、拿不到原始粒子图像，就不要承诺 image-layer dewarping；改做 velocity-field error propagation 和不确定度报告。
- 如果只能做“调包训练网络”，没有物理 forward model，暂时放弃。
- 如果和何远哲正在做的 BOST/NeRIF/PIV/4D 没有接口，暂时只放在背景。
- 如果题目来自 TAS、光谱仪、全息、铁颗粒或等离子体方向，先问清是否由何远哲继续带；若没有师兄数据和明确接口，只作为开题“课题组全景”材料，不转主线。
- 如果外部 BOS 应用只是在氢喷流、微火箭、冷却通道或燃烧速度上换场景，但不能产出可给 He/BOST 使用的数据接口、位移质量控制或误差图谱，就只写进引言/related work，不单独立题。
- 如果 snapshot RI tomography 或硬件并行层析没有可获得数据、代码或明确硬件接口，只当“折射率层析邻居方法”阅读，不承诺复现实验。
