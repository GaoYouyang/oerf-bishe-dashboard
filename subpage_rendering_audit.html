路由规则

  1. 站内 Markdown、JSON、CSV、BibTeX、代码和配置文件默认进入 document_reader.html，并显示目录、统计卡、链接摘要和搜索。
  2. 站内公开 PDF 和图片默认进入 asset_viewer.html；如果 PDF 属于论文库，会自动显示论文上下文、术语导读和论文详情入口。
  3. 论文库每个已缓存公开 PDF 都有一个单篇术语导读页；从术语页点击页码会回到 PDF 包装页对应页。
  4. 选题详情、论文详情、学习库、审计页和桥梁页都是专题页面；全站路由会给普通文本链接加轻量类型标识，帮助判断“点进去是什么”。
  5. 外部 DOI、出版社、GitHub、YouTube/B 站和数据 supplement 不能由本站重渲染，但会被标注为外部来源并保留官方跳转。

边界

公开 GitHub Pages 不放学校 VPN-only 或订阅全文 PDF。私有库、临时下载和 WebVPN 路径会被路由器排除；本页只检查可公开分享的站内资源。

最近一次本地静态覆盖检查：227 个公开 HTML 页面均包含全站链接路由脚本；本页用于快速人工点检代表性入口。
