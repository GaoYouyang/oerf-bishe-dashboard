传统层析 Baseline 到 NeRIF/BOST 的桥梁

这页回答一个导师和师兄都可能会问的问题：如果不用 NeRIF，传统 BOST/CT 重构应该和谁比？答案不是泛泛说“传统方法”，而是明确列出 ART/SART/FBP/TV/LSQR/ASTRA/scikit-image 这些可引用、可解释、可实现的 baseline 层级。

总判断

给师兄看的说法：NeRIF 的对照不应该只写“voxel baseline”。更清楚的写法是：传统线性层析可以按 FBP / ART-SART / regularized least squares / TV minimization 分层；本科 demo 可以先用 scikit-image 跑 Radon + iradon_sart，再视数据复杂度迁移到 ASTRA 或自写 forward/backprojection。

BOST 和医学 CT 的物理观测不同，但数学上都会落到“有限投影下恢复空间场”的反问题。传统 baseline 的价值不是抢主线，而是帮你证明 NeRIF 的提升来自 sparse-view、regularization、continuous-field prior 或 forward model，而不是拿一个过弱的对照。

根文献与工具怎么用

入口 该抽什么 和 NeRIF/BOST 的关系 本地/在线入口
ART, Gordon/Bender/Herman 1970 把投影方程写成线性系统，并用逐投影校正迭代求解。 解释 voxel algebraic reconstruction 的源头；适合写“传统离散场 baseline”。 DOI
SART, Andersen/Kak 1984 同时更新、权重归一、relaxation 和 sparse-view 行为。 最适合本科 demo 的传统 baseline 名称；Open BOS/NIRT 文献也常把 SART/ASART 作为对照。 Purdue PDF / DOI
Kak & Slaney textbook Radon、投影切片定理、FBP、ART/SART、采样和分辨率。 物理本科补 tomography 直觉的首选教材；读相关章节，不用通读。 官方在线目录
Sidky/Kao/Pan TV-min CT 2011 constrained TV minimization，数据一致性和正则化的取舍。 少视角/低信噪下，TV 是神经场之前最自然的强 baseline 之一。 本地 PDF / DOI
Chambolle-Pock 2011 TV/凸成像问题的 primal-dual 优化框架。 如果真的实现 TV baseline，可以引用它解释优化器，而不是只写“加 TV loss”。 HAL PDF / DOI
ASTRA Toolbox GPU projection/backprojection，FBP/SIRT/SART/CGLS 等算法接口。 如果 M1/M2 从 toy 升级到更真实投影几何，ASTRA 是可落地工具。 CWI PDF / DOI
scikit-image Radon / inverse Radon / SART 示例和 Python 图像处理基础。 最快跑通 2D tomography toy；适合 Week 4-6 补反问题与 baseline。 本地 PDF / PeerJ
LSQR, Paige/Saunders 1982 稀疏线性 least-squares 与 Krylov 迭代。 给 CGLS/LSQR baseline、semi-convergence 和 stopping rule 一个正式引用。 DOI

建议的 Baseline 层级

层级 推荐实现 能说明什么 不要夸大的地方
B0 FBP / inverse Radon scikit-image radon / iradon，用于 2D phantom。 让你理解投影、角度、滤波反投影和少角度 artifact。 BOST 不是医学 CT；FBP 只是教学和 sanity check。
B1 SART / voxel iterative scikit-image iradon_sart 或 ASTRA SART/SIRT。 给 NeRIF 一个公平的传统迭代 baseline。 真实 BOST 几何、位移场和 refractive-index conversion 仍需单独处理。
B2 regularized least squares 自写 Ax=b + Tikhonov/LSQR/CGLS，小规模先跑。 把 noise、condition number、semi-convergence 和 stopping rule 写清楚。 不要把过小 toy 的结论当作真实 3D 火焰结论。
B3 TV / sparse-gradient baseline 2D/3D phantom 上做 TV regularization 或 primal-dual 简化版。 和 NeRIF 比较“平滑先验 vs 连续神经场先验”的差异。 TV 会偏好分段常数，火焰/折射率场不一定满足。
B4 NeRIF / neural field coordinate MLP + encoding + BOST projection loss。 主线贡献：连续场、稀疏视角、重投影一致性、可做 encoding ablation。 不能只和弱 baseline 比；需要说明传统 baseline 参数已合理。

最小执行路线

  1. 用 scikit-image 做一个 2D Gaussian / flame-like phantom：生成 30、15、9、5 个投影角。
  2. 跑 FBP、SART、Tikhonov/LSQR、TV 简化版，输出 L2、SSIM、重投影残差和 artifact 图。
  3. 把同一 phantom 交给 M1/M2 的 coordinate MLP / Fourier encoding 版本，比较少视角和噪声下的边界。
  4. 把结论写成一句话：NeRIF 的优势在哪些 view/noise/field-complexity 条件下出现，哪些条件下传统方法已经足够强。
  5. 如果师兄给真实数据，再把 toy 中的 A 和 projection operator 替换为 OERF 几何/位移场接口。

下次问何远哲

  • NeRIF 论文或组内代码的传统 baseline 具体是哪一种：UBOST、SART、TV、RBF-BOST，还是组内体素法？
  • baseline 参数是否有固定设置，例如 relaxation、iteration count、TV weight、early stopping？
  • 真实 BOST 数据更应该比较 field error、reprojection residual、density/temperature error，还是 visual quality？
  • 如果我先做 scikit-image/ASTRA 的 toy baseline，师兄是否认为对开题有帮助？
  • 是否有组内可公开的 phantom 或 synthetic case，用来避免 baseline 只停在通用 CT？

写作边界

不要写成：医学 CT 算法可以直接当作 BOST 算法。

更准确的写法是：ART/SART/TV/LSQR 提供了有限投影反问题的 baseline 语言和实现工具；BOST/NeRIF 的特殊性在于观测量来自背景图案位移/偏折，forward model、标定、折射率-密度-温度链路和真实实验误差都不同。
