科研智能体

Agent for Science / scientific agent

把大语言模型、工具调用、代码执行、实验规划和报告生成连接成科研辅助流程。

读这篇时看什么：蔡老师主页列出 Agent for Science；对你的毕设最稳的落点是 BOST/NeRIF 参数扫描、数据质量检查和自动报告，而不是让 agent 直接替代物理反演。

易混点：agent 生成的结论必须回到真实数据、重投影误差、baseline 和师兄确认；不能把自动报告当作实验事实。

闭环实验

Closed-loop experimentation

根据上一轮实验结果自动决定下一轮参数，使有限实验预算尽量探索最有价值区域。

读这篇时看什么：本科可做成参数扫描助手：先跑粗网格，再按误差/不确定度挑最值得细扫的视角数、噪声水平或网络容量。

易混点：如果目标函数错了，闭环会很高效地优化错误目标；BOST 中必须同时看重投影误差、真值误差和物理可解释性。

风险-覆盖率曲线

Risk-coverage curve / selective prediction

按模型不确定性从高到低拒绝一部分样本，观察保留样本比例下降时平均误差是否同步下降。

读这篇时看什么：它检验 T16 的 reliability score 是否真的能识别失败，而不只是给每个结果附一个没有含义的 confidence。

易混点：只报 AUROC 或平均不确定性不足以证明可用；还要预先定义拒答成本、覆盖率和真实部署阈值。

折射率场

Refractive index field, n(x,y,z)

空间中每一点的折射率分布，决定光线在流场中如何弯折。

读这篇时看什么：NeRIF 最直接的输出对象通常就是折射率场；后续可通过 Gladstone-Dale 关系换算密度。
