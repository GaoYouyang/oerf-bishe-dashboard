Regularization

奇异值分解

Singular Value Decomposition, SVD

把矩阵写成 A = U Sigma V^T。奇异值说明不同方向被观测放大或压低多少，V 中对应零奇异值的方向构成零空间。

读这篇时看什么：用小型 BOST forward matrix 的奇异谱判断可观测 rank、噪声放大和 exact null headroom。

易混点：小 synthetic 矩阵的 SVD 结论不自动代表真实 cone-ray 系统；大规模系统也通常不能直接做完整 SVD。

光流

Optical flow

估计两帧图像间每个像素的运动或位移，是 BOS 位移估计的重要方法之一。

读这篇时看什么：你可以做 BOS 位移估计 benchmark，比较传统相关法、Horn-Schunck、Lucas-Kanade 和现代光流。

伴随算子

Adjoint operator

若 forward operator A 把体场映到观测，伴随 A* 把观测残差按内积意义反传回体场。离散实矩阵下常对应转置 A^T，但它通常不是逆算子。

滤波反投影

Filtered backprojection, FBP

先在投影域做频率滤波，再把各角度投影反投影回空间的解析层析方法。它快、透明，适合作为第一条传统 baseline。

读这篇时看什么：在 2D Radon toy 中比较 FBP 与 SART，观察少视角条纹、噪声放大和正则化需求。

易混点：FBP 的标准公式依赖特定直线积分几何；不能直接把 CT 实现当成真实 BOST 求解器。

病态性

Ill-posedness

解可能不存在、不唯一，或者对噪声极端敏感，这是少视角层析的核心困难。

读这篇时看什么：你的鲁棒性分析、少视角扫描和正则化设计都可以围绕病态性展开。

易混点：不要把“网络能输出一张图”误当成“问题已经有唯一正确解”。

少视角重构

Sparse-view reconstruction

相机视角数量有限时进行三维重构，信息不足导致伪影和不稳定。

读这篇时看什么：非常适合本科切入：3/5/7/9 视角扫描、噪声扫描和 baseline 对比都能形成清晰图表。

逆犯罪

Inverse crime

用同一个离散 forward 生成训练/测试观测，又用它做重建与验收，会让模型在过于匹配的世界里显得异常准确。

读这篇时看什么：T16 必须用不同分辨率、ray tracer、扰动或真实装置做 independent-forward audit。

易混点：随机切分 phantom 并不能消除 inverse crime；forward 离散、几何、噪声与标定也要改变。
