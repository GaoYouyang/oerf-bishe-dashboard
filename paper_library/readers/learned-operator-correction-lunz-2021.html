伴随算子

Adjoint operator

若 forward operator A 把体场映到观测，伴随 A* 把观测残差按内积意义反传回体场。离散实矩阵下常对应转置 A^T，但它通常不是逆算子。

读这篇时看什么：BOST 中可用伴随/反投影形成 physics lift、梯度方向或传统迭代重建的更新。

易混点：A* y 只是把信息摊回去，不等于恢复真实 x；少视角、缩放和几何权重都会影响结果。

算子学习

Operator learning

学习一类输入函数到输出函数的共同映射，而不是只拟合一个固定样本。对 BOST，可把多视角位移/投影函数映射到三维折射率函数。

读这篇时看什么：T16 把它落成跨 phantom、视角、噪声和分辨率共享的 BOST inverse operator；新观测原则上只需一次前向推理。

易混点：算子学习不等于普通 PDE 数值离散，也不等于给每个新样本重新训练一个 NeRIF；是否真的跨网格和跨工况泛化必须实验验证。

神经修正算子

Neural correction operator

先用传统或短迭代物理求解器得到粗重建，再由跨样本学习的算子修正，而不是从观测直接黑箱映射到最终场。

读这篇时看什么：它为 T16 的 physics lift、support-fit 和 learned correction 提供最近的组件级方法先例；可与 direct inverse operator 做公平对照。

易混点：EIT、MRI 或 CT 的修正算子结论不能直接迁移 BOST；粗解误差结构、forward operator 和数据一致性都不同。

病态性

Ill-posedness

解可能不存在、不唯一，或者对噪声极端敏感，这是少视角层析的核心困难。

读这篇时看什么：你的鲁棒性分析、少视角扫描和正则化设计都可以围绕病态性展开。

易混点：不要把“网络能输出一张图”误当成“问题已经有唯一正确解”。

相对 L2 误差

Relative L2 error

常写为 ||x_hat-x||_2 / ||x||_2，用真值能量归一化总体场误差，便于跨样本比较。

读这篇时看什么：作为 synthetic BOST 体场主指标之一，并同时报告逐样本 paired delta、p10/CVaR、harm rate、梯度和积分量。

易混点：单一平均 L2 会掩盖薄前沿、局部峰值、符号错误和少数严重失败；真实数据无 field truth 时也无法直接计算。

奇异值分解

Singular Value Decomposition, SVD

把矩阵写成 A = U Sigma V^T。奇异值说明不同方向被观测放大或压低多少，V 中对应零奇异值的方向构成零空间。

读这篇时看什么：用小型 BOST forward matrix 的奇异谱判断可观测 rank、噪声放大和 exact null headroom。

易混点：小 synthetic 矩阵的 SVD 结论不自动代表真实 cone-ray 系统；大规模系统也通常不能直接做完整 SVD。
