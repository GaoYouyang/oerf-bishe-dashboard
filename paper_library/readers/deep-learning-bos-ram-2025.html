全术语检索

背景纹影

Background-Oriented Schlieren, BOS

用已知背景图案作为参考尺。流场中的折射率梯度会让光线发生微小偏折，最终表现为背景图像的像素位移。

读这篇时看什么：这是 BOST、PIV-BOST 和很多系统误差分析的入口。算法通常先处理 BOS 图像位移，再进入三维反演或速度补偿。

易混点：BOS 直接测到的是图像位移或偏折效应，不是速度场、温度场或密度场本身。

奇异值分解

Singular Value Decomposition, SVD

把矩阵写成 A = U Sigma V^T。奇异值说明不同方向被观测放大或压低多少，V 中对应零奇异值的方向构成零空间。

读这篇时看什么：用小型 BOST forward matrix 的奇异谱判断可观测 rank、噪声放大和 exact null headroom。

易混点：小 synthetic 矩阵的 SVD 结论不自动代表真实 cone-ray 系统；大规模系统也通常不能直接做完整 SVD。

折射率场

Refractive index field, n(x,y,z)

空间中每一点的折射率分布，决定光线在流场中如何弯折。

读这篇时看什么：NeRIF 最直接的输出对象通常就是折射率场；后续可通过 Gladstone-Dale 关系换算密度。

易混点：折射率变化很小，数值尺度容易被忽略；画图时常显示 n-1 或归一化扰动。

Temperature field, T(x,y,z)

读这篇时看什么：BOST 可以在一定假设下由密度推温度；PLIF、TAS、LII 等则可能直接或间接提供温度信息。

光流

Optical flow

估计两帧图像间每个像素的运动或位移，是 BOS 位移估计的重要方法之一。

读这篇时看什么：你可以做 BOS 位移估计 benchmark，比较传统相关法、Horn-Schunck、Lucas-Kanade 和现代光流。

易混点：光流估计的是图像亮度模式运动，不等于物理粒子速度；BOS 中还需和折射模型连接。

背景图案

Background pattern

BOS 中用于测量图像位移的纹理背景，可以是随机点、棋盘、投影图案或自然复杂背景。

读这篇时看什么：位移估计精度与背景纹理强相关；做低成本 BOS demo 时这是第一批要设计的东西。

易混点：图案太稀、太规则或对比度不足都会让相关/光流估计失效。
