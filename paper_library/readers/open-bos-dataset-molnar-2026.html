背景纹影

Background-Oriented Schlieren, BOS

用已知背景图案作为参考尺。流场中的折射率梯度会让光线发生微小偏折，最终表现为背景图像的像素位移。

读这篇时看什么：这是 BOST、PIV-BOST 和很多系统误差分析的入口。算法通常先处理 BOS 图像位移，再进入三维反演或速度补偿。

易混点：BOS 直接测到的是图像位移或偏折效应，不是速度场、温度场或密度场本身。

重投影误差

Reprojection error / projection consistency

用重构出的三维场重新生成各视角观测，再与真实测得的位移或投影比较。

读这篇时看什么：本科项目里它是最容易说清楚的可信度指标：重构得好不好，至少要能解释原始观测。

易混点：重投影误差低不一定代表三维真值准确；病态反问题可能有多个场解释同一组投影。

ART / SIRT / SART

Algebraic reconstruction techniques

把层析重构离散成线性方程组后，用迭代方式求解的经典体素方法。

读这篇时看什么：它们是 NeRIF 论文里最自然的传统 baseline。你要能解释为什么神经场可能比体素 baseline 更连续或更稳。

易混点：传统 baseline 不等于落后；它们可解释、稳定、容易做误差分析，是答辩时的参照物。

伴随算子

Adjoint operator

若 forward operator A 把体场映到观测，伴随 A* 把观测残差按内积意义反传回体场。离散实矩阵下常对应转置 A^T，但它通常不是逆算子。

读这篇时看什么：BOST 中可用伴随/反投影形成 physics lift、梯度方向或传统迭代重建的更新。

易混点：A* y 只是把信息摊回去，不等于恢复真实 x；少视角、缩放和几何权重都会影响结果。

条件数

Condition number

常用最大非零奇异值与最小非零奇异值之比衡量反演对观测扰动的敏感程度。比值越大，微小噪声越可能被放大。

读这篇时看什么：比较不同相机布局、视角数和标定扰动下的线性化 BOST forward 健康度。

易混点：条件数只描述局部线性与所选范数；正则化、非线性 ray bending 和先验会改变实际误差。

物理约束神经网络

Physics-Informed Neural Network, PINN

把物理方程、边界条件或守恒关系写入 loss，让网络不仅拟合数据，也符合物理。

读这篇时看什么：BOS/BOST 后续可从密度场进一步推速度、压力或温度场时，PINN 是重要方法邻居。

易混点：PINN 不自动保证正确；物理方程、尺度归一化和 loss 权重很容易让训练失败。

背景图案

Background pattern

BOS 中用于测量图像位移的纹理背景，可以是随机点、棋盘、投影图案或自然复杂背景。

读这篇时看什么：位移估计精度与背景纹理强相关；做低成本 BOS demo 时这是第一批要设计的东西。

易混点：图案太稀、太规则或对比度不足都会让相关/光流估计失效。

背景纹影层析

Background-Oriented Schlieren Tomography, BOST / TBOS

从多个视角的 BOS 位移场反演三维折射率、密度或温度场，本质是光学投影测量加三维反问题。

读这篇时看什么：这是何远哲 NeRIF、4D BOST 和 PIV-BOST 补偿共同依赖的主技术。你的题名最好保留 BOST/TBOS 或 refractive-index reconstruction。

易混点：完整 BOST 不只是神经网络，还包括相机几何、背景标定、位移估计、正则化、单位换算和重投影验证。
