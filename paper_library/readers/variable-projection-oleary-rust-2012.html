奇异值分解

Singular Value Decomposition, SVD

把矩阵写成 A = U Sigma V^T。奇异值说明不同方向被观测放大或压低多少，V 中对应零奇异值的方向构成零空间。

读这篇时看什么：用小型 BOST forward matrix 的奇异谱判断可观测 rank、噪声放大和 exact null headroom。

易混点：小 synthetic 矩阵的 SVD 结论不自动代表真实 cone-ray 系统；大规模系统也通常不能直接做完整 SVD。

穆尔-彭若斯伪逆

Moore-Penrose pseudoinverse

当 A 不可逆或不是方阵时，A+ 给出最小二乘意义下的最小范数解。它能拟合可观测部分，但不会凭空恢复零空间信息。

读这篇时看什么：建立小矩阵 inverse baseline、定义 range/null projector，并核对 closed-form query update。

易混点：直接对很小奇异值取倒数会放大噪声；真实反问题通常还需要截断或 Tikhonov 正则化。
