Agent for Science 辅助 BOST Pipeline

蔡老师主页把 Agent for Science 列为研究方向之一，但你的毕业设计不应把它做成脱离物理诊断的泛自动化。最稳的落点是：让智能体服务何远哲主线，辅助 BOST/NeRIF/PIV-BOST 的文献整理、参数扫描、质量控制、图表生成和学习日志。

总判断

可以做：把 Agent for Science 做成 BOST 研究工作流助手，帮你自动排参数、跑 M0-M3 demo、检查重投影误差、整理结果表、生成学习日志和师兄沟通材料。

不建议做：把毕设题目改成“全自动科研智能体”本身。这样会离 OERF 光学诊断和何远哲 BOST/NeRIF 主线太远，且本科阶段很难证明科学有效性。

和主线的关系

主线任务 Agent 可以帮什么 必须由物理/算法验证的部分
A: NeRIF / 折射率场重构 自动生成不同视角数、编码方式、loss 权重、网络容量的实验配置；汇总 L2 / SSIM / 重投影误差曲线。 forward model、折射率场定义、单位换算、reprojection error 和 baseline 对比。
B: PIV-BOST 补偿 组织 synthetic particle image pair、OpenPIV/PIVlab 输出、折射畸变前后误差表；标记失败案例。 PIV 图像层/位移层/速度层三层接口，速度误差定义，真实同步数据权限。
C: 4D BOST / 时序低秩 批量跑 rank、frame count、noise level、temporal smoothness；自动生成 rank-error-time 图。 时序一致性指标、低秩假设边界、动态 phantom 或真实高速数据解释。
F: BOS 位移质量控制 对背景图案、光流/互相关、置信图、mask 和异常视角做自动健康报告。 位移不是速度；BOS displacement 与折射率/密度反演之间必须保持物理链条。

本科可执行版本

最小毕业版

  • 固定 M0/M1/M2 或 M3A 一个 demo。
  • 写一个配置扫描器，自动改 view count、noise、loss weight、network size。
  • 每次运行保存 `config.json`、指标表、关键图和失败原因。
  • 生成一页 HTML/Markdown report，供你写学习日志和找师兄讨论。

升级版

  • 加入 active learning / closed-loop：根据上一轮误差挑下一组参数。
  • 加入 input health report：检查图像对、mask、位移场、视角缺失和异常值。
  • 加入论文阅读助手：从本地公开 PDF 的术语页码索引生成精读清单。
  • 加入会前报告：自动整理“这周结果、失败点、下周需要师兄给的数据”。

安全边界

Agent 可以读公开论文库、公开 PDF、你自己的 demo 输出和学习日志。WebVPN/订阅 PDF、账号密码、私有下载目录和未公开组内数据不能进入公开 GitHub Pages，也不能让 agent 自动上传。

Agent 生成的任何科学结论都必须至少经过三层检查：数据来源是否可信、指标是否覆盖目标、图像/重投影是否能解释原始观测。

支撑文献怎么读

文献 读它拿什么 在本项目里的位置
Transforming Science with LLMs 科研生命周期：检索、实验、内容生成、图表、评价和诚信风险。 给 Agent 支撑线总览，适合先读。
The AI Scientist 自动想题、写代码、跑实验、写论文和自动评审的完整 pipeline。 作为反面边界：本科不要承诺全自动发现，只借自动化框架。

Science acceleration and accessibility with self-driving labs 闭环实验、自驱动平台、自动化与可及性。 把 BOST 参数扫描写成 self-driving / closed-loop workflow 的背景。
Autonomous chemical research with large language models LLM 调用工具、规划实验、安全边界。 借 tool-use 语言，不迁移化学实验结论。

可以问何远哲的问题

问题 为什么问
如果我做 Agent 辅助层，师兄更希望它服务 NeRIF 参数扫描、PIV-BOST 数据检查，还是 4D BOST rank/frame-count 扫描？ 防止做出看起来好玩但对组内工作没用的工具。
组内 BOST/PIV-BOST 数据是否已有固定目录结构、命名规则、mask 和 calibration 文件？ Agent 最适合先做数据接口和健康报告。
师兄会不会接受“自动生成周报 + 参数扫描报告”作为毕业设计附加成果？ 确认它是加分项还是偏题项。
哪些指标是组内最在意的：reprojection error、density/temperature error、PIV velocity error、runtime，还是稳定性？ 闭环优化必须有正确目标函数。
Agent for Science is scoped here as a BOST/NeRIF workflow assistant, not as a replacement for optical diagnostics, inverse modeling, or experimental validation.
