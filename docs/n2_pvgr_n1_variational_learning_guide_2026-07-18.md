## 0. 先知道我们在解决什么问题

BOS（Background-Oriented Schlieren，背景纹影）通过观察背景图案的位移，反推出光线经过流场后发生了多大的偏折。BOST（Background-Oriented Schlieren Tomography）进一步从多个观察方向的投影信息重建三维折射率场，进而间接研究密度、温度或反应流结构。

在最简单的模型里，光线被当成一条直线。这个模型便宜、容易放进神经重建程序，但当折射率变化较强时，真实光线会弯曲。于是我们需要区分两个前向测量算子：

- `M`：沿参考直线计算的弱偏折测量；
- `H`：沿真实曲线光线计算的全非线性测量。

当前 N1 候选的核心问题是：能不能复用已经计算过的直线光线状态，用一个低成本的变分模型预测 `H-M`，再决定哪些光线值得用高保真 `H` 重算？这首先是一个“前向算子近似与风险路由”问题，之后才可能成为三维重建或算子学习问题。

## 1. 两个测量算子：`M` 与 `H`

### 1.1 直线弱偏折算子 `M`

设折射率场为 `n(r)`，光线从位置 `a` 出发，沿固定单位方向 `d0` 传播：

```text
r0(s) = a + s d0
```

弱偏折近似把光线轨迹固定为 `r0(s)`，只在这条线上累积折射率梯度对光线方向的横向作用。令

```text
g = grad(log n),       P0 = I - d0 d0^T
```

则典型的直线模型包含如下积分结构：

```text
M  ~  integral P0 g(r0(s)) ds
```

这里的 `~` 表示具体测量还可能包含相机投影、背景板距离、像素尺度、符号约定和窗口界面等几何因素。不要把这个积分直接等同于所有实验装置的最终像素位移。

物理直觉是：折射率梯度像一个“把光线推向一侧的局部力”，弱偏折模型假设光线还没有明显偏离原来的直线路径，因此仍在原来的位置采样梯度。

### 1.2 全非线性曲光线算子 `H`

用弧长 `s` 参数化真实光线，位置 `r(s)` 和单位切向方向 `d(s)` 满足几何光学方程：

```text
r' = d
d' = F(r, d) = (I - d d^T) grad(log n(r))
```

全非线性算子 `H` 沿这条由 ODE 积分得到的弯曲轨迹取样，并把最终方向或出射点映射为 BOS/BOST 的观测量。实际实现常用 RK4 等数值积分；同时还要处理方向归一化、越界、背景平面交点和相机投影。

`H` 比 `M` 贵的原因不只是“多算几个神经网络层”：它每一步都要更新位置和方向，之后在新位置继续查询场，查询序列还可能带来大量调度、验证和自动微分开销。

### 1.3 为什么常见地有 `H-M = O(alpha^2)`

令 `alpha` 表示偏折强度的小参数，例如折射率梯度幅度、密度扰动幅度或无量纲光线偏转量。对平滑折射率场，在固定传播距离、没有光线交叉和可见性突变时：

1. 直线模型中的方向变化通常是一阶量：`delta d = O(alpha)`。
2. 因此曲线相对直线的位置偏移为 `delta r = O(alpha)`。
3. 真实算子在偏移后位置采样梯度，可以作 Taylor 展开：

```text
g(r0 + delta r) = g(r0) + Jacobian(g) delta r + ...
```

由于 `g = grad(log n)`，这里的 `Jacobian(g)` 就是 `Hessian(log n)`，不是 `Hessian(g)`。

4. “位置偏移 `O(alpha)`”乘以“折射作用 `O(alpha)`”后，轨迹位置对测量产生的第一项差异通常是 `O(alpha^2)`。

所以在适用条件下，`H-M` 的主导量常表现为二阶偏折残差。这里的“通常”非常重要：它不是无条件定理。若存在折射率间断、焦散、背景映射奇异、光线穿出域、有限孔径模糊或非静态流场，Taylor 展开和同一测量映射可能失效。N1 必须用不同 `alpha` 做斜率实验，检查 `||M||` 是否约为 `alpha^1`、`||H-M||` 是否约为 `alpha^2`，而不能只引用这个阶数。

## 2. 非齐次变分 defect system 在说什么

把直线参考写成

```text
r0(s) = a + s d0,       d0' = 0
```

令真实曲线相对参考线的状态差为

```text
u(s) = r(s) - r0(s)
v(s) = d(s) - d0
```

沿参考线对真实光线 ODE 做一阶展开，得到候选的非齐次变分系统：

```text
u' = v
v' = F0 + A u + B v
```

其中

```text
F0 = P0 g0
A  = P0 Hessian(log n)
B v = P0[-(d0^T g0) v - d0 (g0^T v)]
```

`g0` 和 Hessian 在 `r0(s)` 上取值，`P0 = I-d0 d0^T`。有些实现会把 `P0` 吸收到 `A/B` 的定义中，或用切空间坐标表达；实现和论文中必须固定同一种约定。

### 2.1 为什么它是“非齐次”的

如果只有 `A u + B v`，系统是齐次的，只描述一个已有扰动如何被传播。这里的 `F0` 是非齐次 forcing：即使初始轨迹和直线完全重合，直线参考下的折射力也会把真实光线推离直线。它是“直线参考并不是精确解”的缺陷源。

可以把它想成：

- `F0`：参考直线每一步欠下的“弯曲账”；
- `A u`：位置已经偏离后，空间折射率梯度变化对后续弯曲的影响；
- `B v`：方向已经偏离后，投影到横向切空间的几何反馈。

因此候选 N1 不是重新从零求解 `H`，而是用参考线上的场信息，近似预测这笔 defect 如何积累。

### 2.2 `A` 项与 `B` 项分别表示什么

**`A u`：空间曲率项。** `A` 主要来自 `grad(log n)` 的空间变化，也就是 Hessian。若光线位置向某个方向偏移，那里看到的折射率梯度会改变多少？纹理窄、火焰面陡或激波附近，`A` 往往更敏感。

**`B v`：方向与投影几何项。** `B` 描述光线方向改变后，横向投影算子和方向与梯度的耦合怎样改变。它不等同于另一个空间 Hessian；即使空间场变化不大，方向偏转也可能改变后续作用方向。

对初学者来说，最重要的不是背公式，而是检查三件事：`A`、`B` 是否保持单位方向约束的切空间结构；数值 JVP 是否与有限差分一致；最终的测量输出是否也线性化了，而不是只修正了出射角。

## 3. 研究定位与边界

N1 可以作为一个有物理依据的候选路线：

> 面向神经 BOST 测量算子，复用直线光线状态，用一阶变分 defect correction 预测曲光线相对弱偏折模型的残差，并在有效域外 fail-closed 回退到高保真求解。

但是下面这些不能现在声称：

- 不能声称首次提出一阶光路扰动、Jacobi/dynamic ray tracing、可微曲光线或多保真抽样；这些方向已有先例。
- 不能把 `A/B` 变分方程本身包装成已证论文创新。
- 没有何远哲现有 TDBOST 实现的输入、目标和前向模型对照，不能声称与其不重叠。
- 没有真实实验数据，不能声称真实 BOST 改善、跨流场泛化、优于 DeepONet/FNO 或论文级加速。
- 没有三阶导数或 Lipschitz 界、标定误差界和有限孔径模型，不能声称“安全证书”已经证明。

真正可能形成论文贡献的组合需要同时通过：物理域有效性、`H-M` 预测误差、三维重建 field relative-L2、逐 rig 尾部误差、成本账本、强基线和真实数据验证。

## 5. 必须掌握的学习顺序

### 第 1 阶段：微积分与线性代数

先补梯度、方向导数、Jacobian、Hessian、Taylor 展开、链式法则、矩阵范数、正交投影和切空间。目标不是会做复杂证明，而是能解释 `grad(log n)`、`A u`、`B v` 每一项的维度和物理含义。

练习：对二维 `n(x,y)=1+alpha f(x,y)` 手算 `grad(log n)`；用有限差分检查 Hessian；验证 `P0 d0=0` 和方向扰动应位于切平面。

### 第 2 阶段：ODE 与数值积分

掌握一阶常微分方程、初值问题、Euler、RK4、局部/全局截断误差、稳定性和步长收敛。然后学习变分方程：对 `y'=f(y)` 做小扰动，理解 `delta y' = J_f delta y`。

练习：让 RK4 步长减半，分别画 `M`、`H` 和 `H-M` 的误差；再对 `alpha` 做 log-log 斜率拟合。不要只看最终视觉图。

### 第 3 阶段：几何光学与 BOS/BOST

掌握折射率、Snell 定律、光程、傍轴近似、折射率梯度、投影几何、背景平面和像素位移。随后读 Raffel，再读 NeRIF 和 OERF 的 TDBOST。

练习：画一条直线光线和一条曲线光线；改变背景距离与梯度幅度，说明像素位移如何改变；列出窗口、有限孔径、流动期间曝光等现实误差。

### 第 4 阶段：逆问题、层析与神经场

学习线性测量 `y=Ax`、欠定问题、正则化、有限视角零空间、可辨识性和训练/测试流场划分。然后学习坐标 MLP、隐式神经表示、自动微分和体积分。

练习：先用固定网格场做三维投影重建，再替换为小型 MLP；比较观测误差和 field relative-L2，理解“投影拟合得好”不一定代表三维场真实。

### 第 5 阶段：PyTorch JVP/HVP 与验证

先掌握 `autograd.grad`、`functional.jvp`、`functional.vjp`，再理解 Hessian-vector product（HVP）不需要显式构造完整 Hessian。对 `A/B`，要做方向有限差分和 JVP dot test：

```text
J v  与  [f(y+eps v)-f(y-eps v)]/(2 eps) 对照
<v, J^T w>  与  <J v, w> 对照
```

所有验证保持 `float64`、固定随机种子，并报告相对误差、越界比例、方向单位范数违反量和实际查询/调用数。MPS/float32 只能用于开发计时，不能自动替代精度审计。

### 第 6 阶段：算子学习与论文实验

在上述前向模型可靠后，再学习 DeepONet、FNO、PINO、神经算子中的 branch/trunk、频域卷积、网格分辨率泛化、函数空间误差和数据划分。N1 可以作为物理前向算子或风险路由模块，不能因为接入神经网络就自动成为“算子学习创新”。

最低实验矩阵应包含：弱偏折 `M`、全非线性 `H`、N1 预测器、一次 Picard 更新、两次 Picard 更新、无路由全高保真、简单阻尼/插值，以及真实可用时的 cone-ray 和实验标定误差。

## 6. 给何远哲的五项最小数据合同

在拿到下面五项之前，N1 只能停留在数学候选和合成开发阶段。请尽量用小规模、可脱敏的样例，不要先要整套实验数据：

1. **当前 TDBOST 畸变模块合同**：输入张量、输出张量、训练目标、损失函数、是否显式使用曲光线，以及一条最小可运行调用链。
2. **真实装置几何合同**：相机数量和位置、相机与背景平面距离、视场/像素尺寸、镜窗或界面、光圈或 f/#、曝光时间。
3. **前向数据合同**：一个小型三维折射率/密度场，至少给出坐标单位、网格间距、边界、`n` 的范围，以及 16 至 64 条固定射线的 `M/H` 可比较结果。
4. **噪声与标定合同**：flow-off 背景、光流/位移噪声量级、相机同步误差、标定误差和可接受的测量不确定度。
5. **评价与发布合同**：实验室认定的重建目标、训练/测试按流场还是按时间划分、field relative-L2/测量误差/尾部误差/运行成本等必须报告的指标，以及哪些数据不能出实验室。

拿到合同后，第一步应是复现师兄已有前向或重建结果；第二步才是把 N1 接入同一评价脚本；第三步才允许讨论论文创新性。

## 7. 当前可执行的判断门

对每个候选版本同时问：

- `H-M` 的预测相对误差是否在不同 `alpha`、不同 rig、不同流场上稳定？
- 预测失败时是否能自动回退，而不是输出看似合理的错误结果？
- 端到端三维场误差是否改善，而不仅是单条光线误差改善？
- 共享状态和导数计算的成本是否低于节省的高保真光线成本？
- 是否公平比较了 DeepONet/FNO 或其他实验室认可基线？
- 结果是否来自独立流场、独立时间片和真实噪声，而非只来自同一合成分布？

只要其中一项没有证据，就把结论写成“待验证”，而不是“有效”。这份指南的目标是帮你把一个值得研究的物理想法，逐步变成可审计的实验；它本身不是论文结论，也不替代导师和师兄对数据合同的确认。
